Rebrand Guide · Site 05 · HVAC

Northline Climate

An HVAC site that finally doesn't look like a 2011 template. Cool navy + cyan palette, animated SVG wind currents in the hero, live seasonal-banner messaging that reads from the visitor's clock, and a three-tier membership plan card.

Files

  • brand.json — brand tokens, palette, webhook.
  • index.html — single-file; all styles + scripts inline. Search data-editable for every changeable string.

Rebrand in 15 minutes

  1. Edit brand.json — name, phone, palette tokens.
  2. Update the 5 CSS custom properties at the top of <style> to match.
  3. Rewrite copy tagged with data-editable.
  4. Adjust the seasonal-banner strings in the inline <script> to match the client's city/climate.
  5. Update the three membership tiers if the pricing structure differs.

Signature techniques

  • Live seasonal banner reads the visitor's clock and swaps messaging by month.
  • SVG wind currents — dashed strokes traveling along bezier paths in the hero.
  • Cyan glow pulse on the seasonal-banner dot.
