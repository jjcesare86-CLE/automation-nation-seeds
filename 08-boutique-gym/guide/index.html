Rebrand Guide · Site 08 · Boutique Gym

Foundry Strength

A brutalist barbell-club site — Anton display, IBM Plex Mono body, red-on-black with a train-timetable schedule board that highlights the actual "next class" based on the visitor's clock.

Rebrand in 15 minutes

  1. Edit brand.json.
  2. Update 5 palette CSS vars.
  3. Replace the six timetable rows with the client's actual schedule (keep the data-hh attribute matching the class time — that's what powers the "NEXT" auto-highlight).
  4. Update the three coach cards and three program cards.

Signature techniques

  • Live "next class in Xh Ym" countdown driven by the visitor's clock.
  • Timetable row auto-flags "NEXT" — the row for the next scheduled class picks up the red accent tint.
  • All-caps Anton display; refunds language is deliberate.
