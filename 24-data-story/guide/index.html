Rebrand Guide · Site 24 · Data Story

Longform: 500 Winters

A scroll-driven cinematic data piece. Five chapters. Each chapter uses a sticky-viz on the left and scrolling narrative on the right; the same 500-year temperature line grows longer as the reader progresses.

Rebrand in 15 minutes

  1. Edit brand.json.
  2. Update the 5 palette CSS variables.
  3. Replace the pseudo-data in the inline script (data array in the viz block) with the client's actual timeseries.
  4. Rewrite the five chapters — kicker, headline, body prose, caption.

Signature techniques

  • Sticky-viz + scrolling narrative pattern (NYT "The Dying Sea" style).
  • Progressive line reveal — each chapter advances the drawn-portion of the timeseries.
  • Scroll-progress bar under the header.
