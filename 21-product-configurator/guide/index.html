Rebrand Guide · Site 21 · Product Configurator

Astria Objects

Real-time product configurator for a small-batch cast concrete table lamp. Drag-to-rotate 3D stage rendered in Canvas 2D (a lightweight substitute for Three.js — no external dependency, no GLB required at seed stage). Three finishes, three stems, live price update. Space-bar toggles auto-spin.

Why Canvas 2D instead of Three.js

Loading Three.js from a CDN adds ~600 KB and network dependency to a page whose whole point is instant interaction. The Canvas 2D substitute is ~150 lines and hits the same UX beats: rotate on drag, swap materials, swap stems, live price, spin toggle. When the seed is sold to a client with a real GLB, swap the draw() function for a Three.js scene using the marker comment in index.html.

Rebrand in 15 minutes

  1. Edit brand.json.
  2. Update 5 palette CSS vars.
  3. Update the three finish options + prices in the finPrice object in the inline script.
  4. Update the three stem options + price deltas in stemPrice.
  5. Swap the fallback hero-lamp.webp for the client's product still.

Signature techniques

  • Drag-to-rotate Canvas 2D lamp with rotation-based lateral light bias.
  • Live-updating price on swatch selection.
  • Keyboard: Space toggles spin, 1–3 switch finish.
