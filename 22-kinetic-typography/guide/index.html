Rebrand Guide · Site 22 · Kinetic Typography

Marrow Type

A showpiece variable-font specimen presented as a creative agency. Roboto Flex driving three live axes (weight, width, slant), a hero that idly breathes, and a work-list where each row transforms font-variation-settings on entrance and hover.

Rebrand in 15 minutes

  1. Edit brand.json.
  2. Update the 3 palette CSS custom properties.
  3. Rewrite the six work-list rows to match real client projects.
  4. Adjust the manifesto to match the agency's actual voice.

Signature techniques

  • Live sliders bound to font-variation-settings — instant demonstration of the axes.
  • Idle-breathing on the hero (~60ms tick) that varies wght/wdth/slnt gently over time.
  • Work-list entrance animation morphs each row from wght:100 wdth:25 → target on IntersectionObserver.
