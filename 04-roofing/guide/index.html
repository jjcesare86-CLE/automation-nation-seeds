Rebrand Guide · Site 04 · Roofing

Summit Roofing Co.

An editorial-quality site for a residential roofing contractor. Cinematic hero, a real before/after slider, and copy that reads like a neighbor wrote it. Rebrandable to any city and any roofer in ~15 minutes.

Concept

This site treats an unglamorous trade with the visual seriousness of a heritage brand. Fraunces italic serif carries the human voice; IBM Plex Sans holds the operating information. Red is used only for CTAs and warranty callouts — never decoratively.

What lives where

  • brand.json — every changeable field: brand name, tagline, phone, address, palette, and FORM_WEBHOOK_URL.
  • index.html — every editable string tagged with a data-editable="…" attribute so a rebranding pass never misses copy.
  • styles.css — five palette variables at the top of :root. Every color in the site references those.
  • app.js — the before/after slider, sticky header, form submit, and the entrance stagger.
  • assets/hero-crew.webp, before.webp, after.webp — swap these three files to make the site the new client's. Aspect ratios: hero 16:9, before/after 3:2.

Rebrand in 15 minutes

  1. Edit brand.json — name, phone, address, palette tokens, webhook.
  2. Update the five palette custom properties in styles.css :root to match.
  3. Replace the three photos in assets/. Keep the file names.
  4. Rewrite the copy in index.html — search for every data-editable attribute; each one names the semantic role (hero.headline, brand.phone, etc.).
  5. Rewrite the three reviews in the .review blocks with real client testimonials and dates. Keep the same length range so the grid stays balanced.
  6. Replace the SVG mountain logomark or leave it — it's built to survive most brands.

Signature techniques

  • Cinematic hero zoom-out on load (14s slow ease-out) — respects prefers-reduced-motion.
  • Draggable before/after slider with pointer, keyboard (← → Home End), and touch support. ARIA slider semantics.
  • Sticky header that solidifies once the hero has scrolled off — reduces visual noise while keeping the phone number one thumb away on mobile.
  • Service cards stagger into viewport at 80ms cadence with a cubic-bezier(.2,.9,.2,1).
  • Form submits to brand.json.FORM_WEBHOOK_URL with mode: 'no-cors' so a GoHighLevel webhook receives it without cross-origin friction.

Copy voice cheat sheet

  • Never use "we're proud" or "trusted." Say what you'll do and by when.
  • Every service ends with a warranty line in monospace — treat it like a signature.
  • Reviews reference street names and dates. Any review that could belong to any roofer isn't a testimonial, it's noise.
  • The word "family" is used once, in the hero. Don't dilute.
