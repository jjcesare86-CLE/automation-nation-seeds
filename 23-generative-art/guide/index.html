Rebrand Guide · Site 23 · Generative Art

Field Study 07

A generative-art collection page. Perlin-noise flow field with a seedable mulberry32 PRNG. Every visit produces a unique canvas piece; the seed is encoded in the URL for rediscoverability. Series of 12 prior seeds shown as thumbnails, each openable.

Rebrand in 15 minutes

  1. Edit brand.json.
  2. Update the 3 palette CSS custom properties.
  3. Optionally change the 5 PALETTES in the inline script to different accent hues.
  4. Update the print sizes + prices in the order form.

Signature techniques

  • Seedable Perlin flow field — same seed produces the same artwork every time.
  • URL as the artwork's provenance — visitors can share their exact piece.
  • Thumbnail chips render at ~200px for a fast archive grid.
