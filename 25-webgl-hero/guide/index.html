Rebrand Guide · Site 25 · Showpiece

Fold — Shader Study

A raymarched liquid-metal WebGL hero. Zero generated assets. Portable — one HTML, one CSS, one JS.

Concept

The full-viewport hero is a signed-distance-field metal blob evaluated per pixel in a single fragment shader. Cursor position feeds displacement; idle time (≥3s no cursor movement) increases displacement so the metal appears to "breathe."

Files that hold this site together

  • brand.json — brand tokens, palette, contact.
  • index.html — copy blocks marked data-editable="hero.headline" etc.
  • styles.css — palette bound to --void, --bone, --mercury, --gold, --dim. All motion easing declared once at the top.
  • app.js — WebGL 1.0. Fragment shader lives inline as a template string. To rebrand, change the three color vec3s near the bottom of main(): mercury, bone, gold.

Rebrand in 15 minutes

  1. Edit brand.json — brand name, tagline, email, palette.
  2. Update the five CSS custom properties in styles.css :root to match.
  3. Update the three vec3s in app.js — mercury, bone, gold — to the same palette.
  4. Rewrite text in the data-editable blocks: hero.headline, hero.sub, study.lead, study.body, studio.about.
  5. Point the mailto link and footer to the new email.

Signature techniques

  • Fresnel-Schlick rim on the SDF for a warm bone-over-mercury metal look.
  • FBM displacement driven by u_mouse, u_time, and an u_idle ramp.
  • Vignette + tone-map + gamma for cinematic finish.
  • Reduced motion — respects prefers-reduced-motion: time freezes, one static frame.
